More to come — but here's the one I keep reaching for: animating to
`height: auto` with a grid row.

```css
.panel {
  display: grid;
  grid-template-rows: 0fr;
  transition: grid-template-rows 0.4s ease;
}

.panel.open {
  grid-template-rows: 1fr;
}
```
